# ThirdPersonPlatformer
## Features
- Third-person movement
- Camera follows the player
- Coins with rotating animation
- UI score counter
- Bonus: Double Jump & Dash

## Controls
- Move: `WASD` / Arrow Keys
- Jump: `Space`
- Dash: `Shift`
